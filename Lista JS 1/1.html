O que é o JavaScript?

É uma linguagem criada para tornar páginas web dinâmicas no lado do cliente, ela é executada no navegador do cliente em um sandbox, o que limita o acesso aos recursos do sistema operacional, fornecendo interfaces predefinidas, a forma de acesso a recursos do sistema se faz por meio de objetos globais disponibilizados pelo navegador, como: document, history, local storage, console etc. Essa linguagem foi criada em 1995, e recebeu esse nome como jogada de marketing para acompanhar a fama da linguagem de programação Java.

O JavaScript é uma linguagem de tipagem dinâmica e forte (Assembly e C tem tipagem fraca), o que a torna muito maleável, mas que também pode causar erros caso não seja tratada corretamente. A tipagem dinâmica também permite que as variáveis sejam atribuídas e reatribuídas a qualquer momento, sem a necessidade de declarar o seu tipo previamente. Os tipos de dados nativos da linguagem são: Number, String, Boolean, Object, Array e Function.
