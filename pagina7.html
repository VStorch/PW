Flexbox

Até 2008 os mecanismos de posicionamento existentes no CSS eram o fluxo do html, float e positions, mas em 2007 se popularizaram os dispositivos móveis. Para possibilitar uma maior responsividade, o CSS 3 trouxe alguns recursos como Screen Media, que permite definir um conjunto de regras diferentes com base no tamanho da tela do usuário, e também a propriedade display:flex, que especifica o mecanismo FlexBox de posicionamento dos elementos HTML

Para definir um componente como um container flex, basta definir o display:flex

        <style>
            .container{display:flex}
        </style>
